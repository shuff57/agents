LiteParse for PDF extraction: Node.js-based tool (not Python), installed via npm. CLI: `npx @llamaindex/liteparse`. Has Python wrapper `liteparse` but it's just a thin wrapper around the CLI. LiteParse outputs plain text headers (e.g., "Chapter 1 Title") not markdown headers (# Chapter 1). Requires post-processing to convert to proper markdown format. OCR requires Tesseract language data (TESSDATA_PREFIX env var). Best for local PDF extraction without API keys.
