# Merge Chapter Reference

## Content Preservation Rules (CRITICAL)
- The Main Text is sacred - preserve 100% exactly as provided
- Do NOT rewrite, summarize, paraphrase, or trim main text
- Preserve all markdown formatting, headers, lists, structure
- Do NOT remove or alter any markdown image links

## Insertion Strategy
- Insert supplemental content BETWEEN existing sections, not within
- Place additional examples after related main-text examples
- Place additional definitions near related main-text definitions
- Maintain logical flow for the student

## Preferred Insertion Points
- After concept explanation → supplemental examples
- After main-text examples → supplemental practice problems
- Before new topic → prerequisite supplemental material
- At section end → supplemental summary or review

## Source Attribution (CRITICAL)
Every Definition, Example, Try It Now, and Problem Set MUST be tagged:
```
**Source:** [Book Name]
```

## Conflict Resolution

| Situation | Action |
|-----------|--------|
| Duplicate definition | Keep BOTH with source tags |
| Conflicting formula | Keep main text; add supplemental as "Alternative" |
| Overlapping examples | Include both if different approaches |
| Same problem, different solution | Include both with source attribution |
